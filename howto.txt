On Ubuntu, first install ruby 1.9.1 and bundler:

    sudo apt-get install ruby1.9.1-full
    sudo gem install bundler

Install all Ruby dependencies for Octopress:

    bundler install


Preview and deploy:

    rake preview      # Locally preview the site on http://localhost:4000
    rake generate     # Generate site in folder public
    rake deploy       # Push all changes to remote master


To resolve issue with outdated version of rake:

    bundler exec rake preview
    bundler exec rake generate
    bundler exec rake deploy
